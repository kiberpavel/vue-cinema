# vue-cinema

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### При запуске
```
cp .env.dist .env
в значение переменной вставте свой url
```
